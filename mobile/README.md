# Market Place (mplace)

Market Place front-end app

## Install dependencies
```bash
yarn
```

### Start app in dev mode
```bash
yarn start
```

### Build app
```bash
yarn build
```

### Start storybook in dev mode
```bash
yarn start:storybook
```

### Build storybook
```bash
yarn build:storybook
```

### Run eslint
```bash
yarn lint
```

### Run eslint with --fix
```bash
yarn lint:fix
```
